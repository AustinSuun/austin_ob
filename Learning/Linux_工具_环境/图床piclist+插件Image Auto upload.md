首先在 Obsidian 插件市场下载 image auto upload
AUR 下载 piclist，配置一下 github 图床，token 在 [github-setting](https://github.com/settings/tokens) 获取，要勾选 repo 权限
自定义域名，使用 jsdelivr 加速
```
https://cdn.jsdelivr.net/gh/AustinSuun/image
```
